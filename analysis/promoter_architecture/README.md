# Promoter Architecture Analysis

This directory contains scripts related to CpG island structure, first exons, TSS overlap, U1-CGI distances, and downstream enrichment analysis.

The `scripts/` subdirectory includes:

- preserved original scripts
- a small number of CLI-friendly helper copies added during repository consolidation

Recommended starting points:

- `scripts/1stexon.py`
- `scripts/CGIcovTSS_sep.py`
- `scripts/u1tocpg_cli.py`
- `scripts/CpG_togeneP_lengthdistribution_cli.py`
