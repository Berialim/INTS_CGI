# Sequencing Processing Pipelines

This directory preserves the original shell-based processing templates for:

- TT-seq
- PRO-seq
- paired-end ChIP-like data
- featureCounts-based quantification

These scripts are intended as lab workflow templates rather than plug-and-play portable pipelines. They assume:

- a prepared reference directory
- installed genomics command-line tools
- an existing sample naming convention
- input FASTQ files placed in an `fq/` directory
